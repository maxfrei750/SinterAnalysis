= SinterAnalysis

Analysis of sinter processes using Mask R-CNN.

== Installation
=== Linux
    git clone --recurse-submodules git@github.com:maxfrei750/SinterAnalysis.git
    cd SinterAnalysis
    conda env create -f environment.yaml

=== Windows
    git clone --recurse-submodules git@github.com:maxfrei750/SinterAnalysis.git
    cd SinterAnalysis
    mklink /D paddle external\paddle\paddle
    conda env create -f environment.yaml
